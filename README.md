# 🚀 ChatBot Pro Dashboard

ChatBot Pro Dashboard is a web-based automation system designed to manage and automate social media messaging for Instagram and Facebook. It provides a centralized dashboard to monitor messages, configure keyword-based auto-replies, and manage communication efficiently.

---

## 📌 Features

* 🔄 **Auto-Reply System** (keyword-based responses)
* 📩 **Multi-Platform Messaging** (Instagram & Facebook)
* 📊 **Real-Time Dashboard Monitoring**
* 🔗 **Webhook Integration**
* ⚙️ **System Status Tracking** (Server, API, Webhook)
* ✉️ **Manual Message Sending Option**
* 🧠 Smart keyword-response management

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/chatbot-pro-dashboard.git
cd chatbot-pro-dashboard
```

### 2️⃣ Install Dependencies

(Depending on your backend)

For Node.js:

```bash
npm install
```

For Python:

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Run the Project

For Node.js:

```bash
npm start
```

For Python:

```bash
python app.py
```

---

### 4️⃣ Open in Browser

```
http://127.0.0.1:5507/dashboard.html
```
